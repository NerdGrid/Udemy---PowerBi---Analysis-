# 📊 Udemy Courses Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard analyzing Udemy courses data. It provides insights into sales, subscribers, course distribution, and content trends.

## 🚀 Key Features
- 📈 Sales Analysis
- 👥 Subscriber Insights
- 📚 Course Distribution
- ⏱️ Content Duration Analysis
- 🔍 Keyword Analysis
- 💰 Paid vs Free Courses

## 📊 Dashboard Highlights
- Total Courses: 3672  
- Total Sales: $881.67M  
- Total Content Duration: 15.05K hours  
- Total Subscribers: 11.72M
